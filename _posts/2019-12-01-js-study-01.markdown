**字符串**

这里比较重要的一个是字符串的拼接：
>var name = '小明';   
var age = 20;  
var message = '你好, ' + name + ', 你今年' + age + '岁了!';   
等于：var message = \`你好, ${name}, 你今年${age}岁了!\`;   
注意这里用的是反撇号`。
